# LLM Reference Documentation

This directory contains compact reference documentation optimized for AI tools and LLM assistants.

## Files

| File | Description |
|------|-------------|
| `convex-llms.txt` | Convex documentation reference with URLs to official docs |
| `daytona-llms.txt` | Daytona SDK documentation reference |
| `daytona-cli-llms.txt` | Daytona CLI command reference |

## Usage

These files are designed to be consumed by AI coding assistants to provide context about the technologies used in this project. They contain:

- Compact summaries of key concepts
- Links to official documentation
- API references and common patterns

Prefer these compact versions when you have web access. For comprehensive offline reference, see the full documentation versions in the project root or official documentation sites.
